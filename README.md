# Pal-ndromo-Ensamblador-EyPC
Programa en lenguaje ensamblador que determina si una cadena de texto es un palíndromo. El algoritmo procesa la cadena eliminando espacios y convirtiendo letras mayúsculas a minúsculas, para después comparar los caracteres desde ambos extremos y verificar si la secuencia es simétrica. Finalmente, muestra en pantalla si la cadena es o no un palíndromo.
